### HWT Hash Web Tokens
Tokens for passing auth state around: sovereign authentication for multi-agent delegation chains with decentralized domain-based authority, incorporates community based authorization conventions for jurisdiction/data sovereignty and schema. [Project](https://github.com/hwt-protocol) includes [protocol](https://github.com/hwt-protocol/hwt-protocol/blob/main/SPEC.md) (separation of concerns), [reference library](https://jsr.io/@hwt/hwtr-js) (JavaScript implementation "Hwtr"), [demos](https://github.com/hwt-protocol/hwt-demo) (Deno, Cloudflare, etc).
